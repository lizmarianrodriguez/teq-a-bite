# Teq-A-Bite — Página Web Premium

## 1. Descripción del Proyecto
Página web premium para Teq-A-Bite, restaurante de cocina venezolana en Kissimmee, Florida. El objetivo es convertir visitantes en clientes a través de un diseño visualmente impactante, inspirado en cadenas modernas como Shake Shack: imágenes grandes, tipografías elegantes, animaciones suaves y la comida como protagonista absoluta.

Público objetivo: Residentes y turistas de Orlando/Kissimmee que buscan comida venezolana auténtica, y personas que quieren probar esta gastronomía por primera vez.

## 2. Estructura de Páginas
- `/` — Homepage (single-page con secciones scrollables)
  - Hero con foto de plato estrella
  - Nuestra Historia
  - Menú visual por categorías (Arepas, Tequeños, Cachapas, Pepitos, Hamburguesas, Bebidas)
  - Galería de fotos
  - Reseñas de clientes
  - Mapa y datos de contacto
  - Footer elegante

## 3. Características Principales
- [ ] Hero con imagen full-width de arepa/plato estrella + overlay oscuro + tipografía elegante
- [ ] Navegación sticky con fondo transparente que cambia al scrollear
- [ ] Sección "Nuestra Historia" con texto inspirador y foto del restaurante
- [ ] Menú visual con tarjetas por categoría + precios
- [ ] Galería de fotos apetitosas en grid
- [ ] Bloque de reseñas con estrellas y citas reales
- [ ] Mapa embebido (Google Maps) + info de contacto
- [ ] Footer elegante con redes sociales y horarios
- [ ] Animaciones suaves (scroll reveal, hover effects)
- [ ] Totalmente responsive

## 4. Modelo de Datos
No se requiere base de datos. Datos estáticos:
- Información del restaurante (nombre, dirección, teléfono, email, horarios)
- Menú con categorías, items y precios
- Reseñas de clientes
- Galería de imágenes

## 5. Integraciones Backend / Terceros
- Google Maps Embed (iframe) para mostrar ubicación
- Reseñas de Yelp como referencia visual (no integración API directa)
- Links a pedidos online (FromTheRestaurant)
- Links a redes sociales (Facebook, Instagram)
- **Ninguna integración de backend requerida**

## 6. Plan de Fases de Desarrollo

### Fase 1: Homepage Premium (Single-Page)
- Goal: Construir la página principal con todos los elementos visuales y contenido completo
- Deliverable: Homepage con Hero, Historia, Menú, Galería, Reseñas, Mapa y Footer

### Fase 2: Animaciones y Optimización
- Goal: Agregar animaciones de scroll reveal, transiciones suaves, y optimización de performance
- Deliverable: Interacciones polish para experiencia premium

### Fase 3: SEO y Metadatos
- Goal: Optimizar SEO local para Kissimmee/Orlando, tags Open Graph, Schema.org
- Deliverable: SEO completo para ranking local
